Projects

Audio Equalizer

This was an individual project that was designed and completed independently. The circuit is able to take an input signal such as audio and separate it into three bands of frequencies: bass, mid, and treble. The circuit allows the user to control the volume of each band of frequency to change the characteristics of the audio. The signal is amplified and outputted into a speaker.

Single Photon Image Sensor

This is an ongoing senior project with a team of six members. The design for the chip is a single photon detector with a frame rate of 1M FPS. The chip is currently being tested on a PCB and with LabView.

Memory Storage Device

This was an individual project. Microcontrollers, PIC and Arduino, were used to interface with a SD card and serial flash chip. Data can be written and extracted from the card by using serial peripheral interface. The project was used to extract the data from a stored audio file and to convey the information using pulse width modulation.

Tetris

This was an individual project. A microcontroller, PIC, was used to control a led matrix to simulate a game of Tetris. The rules, user interface, and algorithms for the game was coded in C language using MPLABX.

Mini Rose Float Parade

The project consisted of fifteen members. The team was divided into smaller groups that worked on a specific aspect of the float. I worked in a group of three that was responsible for the animations of the rose float.
